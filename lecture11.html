Lecture 11 – ES6+ Features

Transcript Summary

  • let/const, template literals, destructuring, spread/rest.
  • Modules (import/export), classes, Maps/Sets.
  • Optional chaining, nullish coalescing, logical assignment.

Highlights

// Destructuring + default + renaming
const user = { name: "Aman", city: "Delhi" };
const { name: fullName, city = "NA" } = user;

// Spread / Rest
const a = [1,2], b = [3,4];
const all = [...a, ...b];
function sum(...nums){ return nums.reduce((x,y)=>x+y,0); }

// Template strings
console.log(`Hello, ${fullName} from ${city}`);

// Optional chaining + nullish coalescing
const profile = { social: null };
const twitter = profile.social?.twitter ?? "not set";

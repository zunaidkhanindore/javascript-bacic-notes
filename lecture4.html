Lecture 4 – Loops (for, while, do...while, for...of, for...in)

LoopBest ForExample
forIndex basedfor (let i=0; i<n; i++)
whileUnknown iterationswhile (cond) {...}
do...whileRun at least oncedo {...} while (cond)
for...ofIterablesfor (const v of arr)
for...inObject keysfor (const k in obj)
Tips
Arrays पर for...in avoid; order unpredictable हो सकता है
Array methods (map, filter, reduce) readability बढ़ाते हैं

Examples

const arr = [10,20,30];
for (const v of arr) console.log(v);

const user = { name: "Aman", age: 20 };
for (const k in user) console.log(k, user[k]);

let i = 0;
while (i < 3) { console.log(i); i++; }

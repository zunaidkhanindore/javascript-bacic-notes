Lecture 7 – Functions in JavaScript

Transcript Summary

  • Functions reusable code blocks हैं; call करने पर चलते हैं।
  • Parameters input लेते हैं; return output देता है।
  • Types: Declaration, Expression, Arrow, IIFE.
  • Default parameters, rest parameters (...args) help करते हैं flexible APIs बनाने में।

Function Forms

FormSyntaxNotes
Declarationfunction sum(a,b){ return a+b; }Hoisted
Expressionconst sum = function(a,b){...}Not hoisted (binding only)
Arrowconst sum = (a,b) => a+b;Lexical this
IIFE(function(){ ... })();One‑off scope

Examples

// Declaration
function greet(name = "User") {
  return `Hello, ${name}`;
}
console.log(greet("Aman"));

// Expression + Callback
const twice = function (fn, x) {
  return fn(fn(x));
};
const inc = n => n + 1;
console.log(twice(inc, 3)); // 5

// Arrow + Rest
const average = (...nums) =>
  nums.reduce((a, b) => a + b, 0) / nums.length;
console.log(average(10, 20, 30)); // 20

this Binding (Arrow vs Regular)

const counter = {
  val: 0,
  inc() { this.val++; },
  incLater() {
    setTimeout(() => { // arrow -> lexical this
      this.val++;
      console.log("val:", this.val);
    }, 0);
  }
};
counter.inc();
counter.incLater();

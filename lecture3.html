Lecture 3 – Conditional Statements (if / else if / else / switch)

PatternUse Case
if (cond) { ... }Single check
if ... elseTwo outcomes
if ... else if ... elseRanges / multiple checks
switch (x) { case ... }Discrete values
Common Pitfalls
Loose equality (==) के बजाय strict (===) use करो
switch में break भूलना → fallthrough
Nested if को small functions में तोड़ो

Examples

// Grading
const marks = 82;
if (marks >= 90) console.log("A");
else if (marks >= 75) console.log("B");
else console.log("C");

// Switch day
const day = 2;
switch (day) {
  case 1: console.log("Mon"); break;
  case 2: console.log("Tue"); break;
  default: console.log("Other");
}

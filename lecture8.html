Lecture 8 – Objects in JavaScript

Transcript Summary

  • Objects key–value pairs का collection हैं; keys string/symbol होती हैं।
  • Shorthand properties, computed keys, methods commonly use होते हैं।
  • Deep copy vs shallow copy का difference समझो।

Create & Access

// Literal + shorthand + computed key
const name = "Aman";
const key = "score";
const user = {
  name,
  age: 20,
  greet() { return `Hi, ${this.name}`; },
  [key]: 95
};

console.log(user.name, user[key], user.greet());
console.log("city" in user); // false

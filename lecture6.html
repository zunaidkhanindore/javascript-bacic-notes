Lecture 6 – Arrays in JavaScript

Introduction

  • Array एक ordered collection होता है जिसमें multiple values store की जा सकती हैं।
  • Arrays में elements का index 0 से शुरू होता है।
  • JavaScript arrays में किसी भी data type के elements हो सकते हैं।

Creating Arrays

// Using array literal
let fruits = ["Apple", "Banana", "Mango"];

// Using new Array()
let numbers = new Array(1, 2, 3, 4);

Accessing Array Elements

CodeOutput
fruits[0]Apple
fruits[1]Banana
fruits[2]Mango

Array Methods

MethodDescriptionExample
push()Adds element at the endfruits.push("Orange")
pop()Removes last elementfruits.pop()
shift()Removes first elementfruits.shift()
unshift()Adds element at startfruits.unshift("Papaya")
includes()Checks if element existsfruits.includes("Apple")
indexOf()Returns index of elementfruits.indexOf("Mango")
join()Joins array into stringfruits.join(", ")

Looping through Arrays

// Using for loop
for(let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

// Using for...of
for(let fruit of fruits) {
    console.log(fruit);
}

Transcript Summary

  • Arrays को store और manage करने के लिए use किया जाता है।
  • Indexing 0 से start होती है।
  • JavaScript arrays dynamic होते हैं — size auto adjust होता है।
  • Built-in methods arrays को modify और manipulate करने में help करते हैं।

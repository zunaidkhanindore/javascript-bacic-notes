Lecture 5 – Strings & String Methods

MethodUsageExample
lengthString size"hello".length → 5
toUpperCase()Upper"hi".toUpperCase()
toLowerCase()Lower"HI".toLowerCase()
includes()Substring check"abc".includes("b")
indexOf()First index"ababa".indexOf("ba")
slice()Substring"hello".slice(1,4)
trim()Edges whitespace" hi ".trim()
replace()Replace first match"a-b".replace("-", "_")
split()To array"a,b".split(",")
Notes
Strings are immutable; methods new string लौटाते हैं
Template literals backticks (`) use करते हैं

Examples

const name = "Aman";
console.log(`Hello, ${name}!`);   // template literal

const msg = "  hi JS  ";
console.log(msg.trim().toUpperCase()); // "HI JS"
